Counting Primes

Problem Statement:

Students should write a parallel c program for “Counting Primes” using the following two methods: MPI_Send and MPI_Receive ONLY

Given

<ul>
<li>– &nbsp;Lower bound number x</li>
<li>– &nbsp;Upper bound number yOutput</li>
</ul>
<ul>
<li>– &nbsp;Count of prime numbers occurring between x and y</li>
<li>– &nbsp;Count of prime numbers occurring in each process.Example:
“Total number of prime numbers is: 20” “Total number of prime numbers in P1 is: 10” “Total number of prime numbers in P2 is: 6” “Total number of prime numbers in P3 is: 4”
</li>
</ul>

Parallelization Scenario Master Process:

<ul>
<li>– &nbsp;Calculate the sub range size r=(y-x)/p</li>
<li>– &nbsp;Note that p is the number of processes.</li>
<li>– &nbsp;Broadcast x and r to each slave process using MPI_Send.</li>
<li>– &nbsp;Receive sub count from each slave process using MPI_Receive.</li>
<li>– &nbsp;Print total count of primes between x and y.Slave Process:</li>
</ul>
<ul>
<li>– &nbsp;Receive x and r through the MPI_Receive.</li>
<li>– &nbsp;Calculate the lower bound a and upper bound b according to its rank.</li>
<li>– &nbsp;Count primes in its sub range (between a and b).</li>
<li>– &nbsp;Print the partial count</li>
<li>– &nbsp;Send this partial count to the master process using the MPI_Send.</li>
</ul>
